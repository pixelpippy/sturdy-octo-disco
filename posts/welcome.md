## What Is VitePress?

VitePress is a static site generator powered by [Vite](https://vitejs.dev/) and [Vue](https://vuejs.org/). It's designed for building fast, content-focused websites. Some highlights:

- **Fast**: Instant server start and hot module replacement during development.
- **Markdown-based**: Write content in Markdown with full Vue component support.
- **Customizable**: Extend the default theme or build your own.
